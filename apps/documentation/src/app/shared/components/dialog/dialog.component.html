Usage Instructions

Add the following import to the appropriate @NgModule: 
import {{ '{' }} {{ icon }} {{ '}' }} from '@ng-icons/{{ library }}';

@NgModule({{ '{' }}
  imports: [
    NgIconsModule.withIcons({{ '{' }} {{ icon }} {{ '}' }})
  ]
{{ '}' }})
In your component, add the following code: 
<ng-icon name="{{ icon | dash }}"></ng-icon>
